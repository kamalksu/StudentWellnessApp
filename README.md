# Welcome to the student wellness app 👋


## Get started
1. Clone the app
   ```bash
   git clone https://github.com/kamalksu/StudentWellnessApp.git
   cd StudentWellnessApp   

3. Install dependencies

   ```bash
   npm install
   ```

4. Start the app

   ```bash
   npx expo start
   ```
